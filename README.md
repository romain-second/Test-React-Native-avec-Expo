# Projet Liste de Prénoms

Mini‑app Expo (sans dépendances externes) avec un champ prénom, un bouton d'ajout et une liste des prénoms saisis.

## Installation

```bash
npm install
```

## Lancement

### Mobile (Android / iOS)

```bash
npm start
```
Scannez le QR code avec l'application **Expo Go**.

### Web

```bash
npm run web
```
L'application s'ouvrira dans votre navigateur.
